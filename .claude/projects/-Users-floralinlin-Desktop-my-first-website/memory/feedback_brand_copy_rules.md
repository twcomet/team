絕對不可改動以下兩句品牌語，任何情況下都必須原文使用：

1. **繪新讓世界煥然一新**
2. **技術翻轉生活，信仰翻轉生命**

**Why：** 佳樺明確宣告這是「天條」——品牌最高準則，不可意譯、不可縮減、不可重新詮釋。

**How to apply：** 出現在任何文案、UI 文字、行銷素材、程式碼字串時，必須完整原文複製。不能用「讓世界煥然一新」（少了「繪新」）、不能用「技術改變生活」（「翻轉」不能換成「改變」）、不能拆開使用其中一句而省略另一句的語義。
