每次任務完成後，依序執行以下協議：

1. **防復發** — 問題會再發生嗎？會就加防護（validation、規範、guard）
2. **跨服務掃描** — 其他頁面/元件有同樣問題嗎？有就一併修
3. **知識沉澱** — 值得生成 skill / memory / rule 嗎？值得就寫進記憶
4. **Git push** — 推了嗎？沒有就推（確保跨裝置同步）
5. **跨裝置** — 我們會跨裝置協作，資料必須可以同步

**Why：** 佳樺明確要求這是標準流程，不只是「完成任務」，而是要確保品質、同步、傳承。

**How to apply：** 每次對話的最後一個任務完成後，自動執行此 checklist，不需要被提醒。
